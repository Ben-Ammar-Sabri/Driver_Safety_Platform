# Driver_Safety_Platform
The Driver Safety Platform is an intelligent AI-powered driver monitoring solution.
It combines an embedded module, advanced AI analysis, an automated n8n workflow, and a web dashboard designed for insurers, fleet managers, and families.

The goal: reduce accidents through continuous monitoring of driver behavior and road conditions.
